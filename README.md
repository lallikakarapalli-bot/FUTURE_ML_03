# Resume Screening System

## 📌 Overview
This project is a simple Resume Screening System that automatically evaluates candidate resumes based on required job skills. It calculates match scores, ranks candidates, and identifies the best candidate for a job role.

## 🚀 Features
- Automated resume screening
- Skill-based candidate matching
- Match score and percentage calculation
- Candidate ranking system
- Top candidate selection

## 🛠️ Technologies Used
- Python
- Pandas
